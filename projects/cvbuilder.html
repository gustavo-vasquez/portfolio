CVBuilder

Asistente detallado para la creación de curriculum en línea de forma rápida y eficaz.

Portada de inicio

En la página principal verás todas las ventajas de usar la aplicación para crear tus curriculum. Se divide en:

  • Plantillas

  • Descargas

Login y registro

En esta pantalla te encontrarás con la posibilidad de registrarte de forma tradicional usando una cuenta de correo y contraseña. Como alternativa puedes utilizar cuentas externas como Github, LinkedIn, Microsoft o Google. Los datos extraidos de esas cuentas se utilizarán automáticamente para rellenar algunos campos del curriculum.

Detalles personales

Cada sección esta agrupada por pestañas donde se completa y guarda la información modularmente. Se puede editar o borrar además de hacer cada bloque visible o no visible (esta característica se aplica también para la sección entera).

Estudios

Cambio de plantilla

Este menú permite alternar entre las tres plantillas disponibles para el curriculum (Classic, Elegant, Modern). La aplicación es automática. Aparece tanto en la pantalla de edición como en la pantalla de finalizado de cv.

Opciones de guardado de CV

Aquí aparecen tres opciones:

  • Imprimir: prepara el documento para imprimir directamente a papel o hacer uso de la generación automática de archivo pdf integrada en los navegadores basados en chromium.
  • Cambiar plantilla: permite alternar entre todas las plantillas disponibles.
  • Descargar como pdf: se prepara el documento para descargar el curriculum como archivo pdf, listo para enviar por correo electrónico.
