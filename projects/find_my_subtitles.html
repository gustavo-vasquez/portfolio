Find my subtitles

Aplicación winforms para buscar subtítulos de series y películas usando como fuente sitios web utilizando web scraping.

Ventana principal

Al abrir el programa podrás elegir buscar subtítulos en cualquiera de las tres fuentes disponibles. Una vez que se hizo la descarga, desde el icono de carpeta se puede abrir el explorador de Windows directamente en la carpeta de descargas.

Búsqueda por subdivx

Al buscar, el programa hará uso de web scraping y recopilará toda la información necesaria desde la página de resultados del sitio web. Según la fuente elegida para buscar subtítulos, los resultados aparecerán de forma diferente.
