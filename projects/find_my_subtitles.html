Find my subtitles

Aplicación winforms para buscar subtítulos de series y películas mediante web scraping a sitios web.

https://github.com/gustavo-vasquez/findmysubtitles-project

Ventana principal

Al abrir el programa podrás elegir buscar subtítulos en cualquiera de las tres fuentes disponibles:

SubDivX

TuSubtitulo

Subtitulamos

Búsqueda de subtítulos

Al buscar, el programa hará uso de web scraping y recopilará toda la información necesaria desde la página de resultados del sitio web. Según la fuente elegida para buscar subtítulos, los resultados aparecerán de forma diferente. En los resultados de búsqueda puedes acceder al enlace directo del subtítulo elegido en la web correspondiente y descargarlo. Si la búsqueda se hace eligiendo Subtitulamos.tv puedes descargar directamente desde el programa.

Programado en:
C#
.NET Framework
